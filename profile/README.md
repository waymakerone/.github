## Waymaker — Productivity you need. Apps you build.

WaymakerOS is a productivity suite you can build on — 20 tools for daily operations, plus a platform to build custom apps, agents, and automations with AI.

**Commander** is the foundation. **Host** is the build layer. **One** is the intelligence.

### Project Docs

Every project starts with structure. **[project-docs](https://github.com/waymakerone/project-docs)** is our open-source documentation framework for AI-assisted development — PRDs, phased build prompts, session briefs, and CLAUDE.md routing. It's the skeleton every blueprint is built on, and it works with any AI coding tool.

```
your-project/
├── CLAUDE.md                              # AI reads this first
├── docs/01-planning/product-requirements/ # What to build
├── docs/02-working/prompts/active/        # Phased build instructions
└── docs/02-working/sessions/              # Session handoff briefs
```

### Blueprints

Blueprints are project packs built on the project-docs framework. Clone one, open it with your AI coding tool (Claude Code, Cursor, Codex), and build a production app on WaymakerOS in minutes.

Each blueprint includes a PRD, phased build prompts, and WaymakerOS integration — everything an AI agent needs to build the app end-to-end.

**Websites** — Restaurant, Portfolio, Professional Services, Real Estate, Retail, Trades, Health & Fitness, Events

**Tools** — Expense Tracker, Simple Invoice, Time Tracker, Leave Manager, Applicant Tracker, Asset Register, Client Portal, Document Signing, Feedback Survey, Service Desk, CRM (B2B & B2C), Team Directory

### Get Started

1. Start any project with [project-docs](https://github.com/waymakerone/project-docs): `git clone https://github.com/waymakerone/project-docs my-project`
2. Or grab a blueprint: `git clone https://github.com/waymakerone/<blueprint-name>`
3. Open it with your AI coding tool
4. Tell the agent: *"Read CLAUDE.md and build this"*
